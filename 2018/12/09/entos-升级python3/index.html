centos 升级python3

发表于 2018-12-09 | 分类于 技术 ， 教程 | | 阅读次数:
安装centos后，需要将自带的python2.7升级到python3.x

首先下载可能用到的依赖

然后下载python3.6.4的安装包
# wget https://www.python.org/ftp/python/3.6.4/Python-3.6.4.tgz
解压
# tar -zxvf Python-3.6.4.tgz
删除之前的python2
# rm -rf /usr/bin/python
新建安装目录
# mkdir /usr/local/python3
进入解压的文件夹，配置安装目录
# ./configure --prefix=/usr/local/python3
安装

创建软连接（可能报错，需要删掉之前的连接）
# ln -s /usr/local/python3/bin/python3 /usr/bin/python
修改yum
# vim /usr/bin/yum
第一行修改为
#!/usr/bin/python2.7
